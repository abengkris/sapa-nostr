# Tell it!

**Whatever it is, just Tell It.**

A decentralized, metadata-resistant microblogging application built on the [Nostr](https://github.com/nostr-protocol/nips) protocol.

Domain: [tellit.id](https://tellit.id)

## Features

- **NIP-17 Private Messaging**: Secure, end-to-end encrypted direct messages using Gift Wraps (Kind 1059).
- **Decentralized**: No central server. Your data lives on the relays you choose.
- **Rich Media Support**: Share images, videos, and links seamlessly.
- **Microblogging**: Post notes, replies, and reactions (likes, reposts).
- **User Status**: Share what you're doing or listening to in real-time.
- **Article Support**: Read and write long-form content using Kind 30023.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Nostr SDK**: NDK (Nostr Dev Kit)
- **Messaging**: @nostr-dev-kit/messages (High-level NIP-17 API)
- **Styling**: TailwindCSS
- **State Management**: Zustand
- **Icons**: Lucide React

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)
